# ohaasa-discord-bot
오하아사 순위를 아침마다 알려주는 디스코드 봇입니다.

┌─────────────────────────────────────────────────────────┐
│ 1. 스케줄러 & 실행 환경 (GitHub Actions - daily_bot.yml)│
└─────────────────────────────────────────────────────────┘
      │ - ⏰ 트리거: 매일 아침 6시 30분 (cron) 또는 수동 실행
      │ - 💻 환  경: Ubuntu 가상 서버 할당
      │ - 📦 준  비: Python, Playwright, Chrome 브라우저 설치
      ▼
┌─────────────────────────────────────────────────────────┐
│ 2. 데이터 수집 / 크롤링 (main.py - Playwright)          │
└─────────────────────────────────────────────────────────┘
      │ - 🌐 접  속: 아사히 방송 오하아사 웹페이지 호출
      │ - 👻 브라우저: 화면 없는(Headless) 크롬 브라우저 백그라운드 실행
      │ - ⏳ 렌더링: 자바스크립트가 데이터를 채울 때까지 대기 (최대 15초)
      │ - 📄 출  력: 데이터가 모두 로딩된 최종 HTML 코드 확보
      ▼
┌─────────────────────────────────────────────────────────┐
│ 3. 데이터 가공 / 파싱 (main.py - BeautifulSoup)         │
└─────────────────────────────────────────────────────────┘
      │ - 🔍 색  인: HTML 내에서 'ul.oa_horoscope_list li' 태그 12개 탐색
      │ - 🔄 변  환: 영문 클래스명(aries 등) ➡️ 한글 별자리명 매핑
      │ - 🔢 정  렬: 읽어온 순서대로 1~12위 순위 및 메달 이모지(🥇🥈🥉) 부여
      ▼
┌─────────────────────────────────────────────────────────┐
│ 4. 데이터 조립 및 전송 (main.py - Requests)             │
└─────────────────────────────────────────────────────────┘
      │ - 🎨 디자인: 추출한 데이터를 Discord Embed(임베드) 형식의 JSON으로 조립
      │ - 🔑 인  증: GitHub Secrets에 숨겨둔 DISCORD_WEBHOOK URL 호출
      │ - 🚀 전  송: 디스코드 서버로 HTTP POST 요청 발송
      ▼
┌─────────────────────────────────────────────────────────┐
│ 5. 최종 목적지 (Discord App)                            │
└─────────────────────────────────────────────────────────┘
        ✨ 내 디스코드 채널에 깔끔한 임베드 메시지 도착!
