# 아침별점 일일 알림

## 설명
오하아사 순위를 아침마다 알려주는 Webhook 기반 단방향 알림형 봇입니다.
- 특정 시간에 아사히 아침별점(おは朝星占い) 홈페이지를 크롤링하여 메세지를 전달합니다.
  - 평일 : https://www.asahi.co.jp/ohaasa/week/horoscope/
  - 주말 : https://www.tv-asahi.co.jp/goodmorning/uranai/

## 파일 구조
```
├── .github               
│   └── workflows        
│       └── daily_bot.yml  # GitHub Actions 설정 파일
└── main.py                # 파이썬 크롤링 코드
```

## 전체 구조도
```
┌─────────────────────────────────────────────────────────┐
  1. 스케줄러 & 실행 환경 (GitHub Actions - daily_bot.yml) 
└─────────────────────────────────────────────────────────┘
      │ - 매일 아침 특정 시간(cron) 또는 수동 실행
      │ - Ubuntu 가상 서버 할당
      │ - Python, Playwright, Chrome 브라우저 설치
      ▼
┌─────────────────────────────────────────────────────────┐
  2. 데이터 수집 / 크롤링 (main.py - Playwright)           
└─────────────────────────────────────────────────────────┘
      │ - 아사히 방송 오하아사 웹페이지 호출
      │ - Headless 크롬 브라우저 백그라운드 실행
      │ - 자바스크립트가 데이터를 채울 때까지 대기 (최대 15초)
      │ - 데이터가 모두 로딩된 최종 HTML 코드 확보
      ▼
┌─────────────────────────────────────────────────────────┐
  3. 데이터 가공 / 파싱 (main.py - BeautifulSoup)          
└─────────────────────────────────────────────────────────┘
      │ - HTML 내에서 별자리 순위 관련 태그 12개 탐색
      │ - 영문 클래스명(aries 등)을 한글 별자리명으로 매핑
      │ - 읽어온 순서대로 1~12위 순위 및 메달 이모지(🥇🥈🥉) 부여
      ▼
┌─────────────────────────────────────────────────────────┐
  4. 데이터 조립 및 전송 (main.py - Requests)              
└─────────────────────────────────────────────────────────┘
      │ - GitHub Secrets에 숨겨둔 DISCORD_WEBHOOK URL 호출
      │ - 디스코드 서버로 HTTP POST 요청 발송
      ▼
┌─────────────────────────────────────────────────────────┐
  5. 최종 목적지 (Discord App)                            
└─────────────────────────────────────────────────────────┘
        내 디스코드 채널에 깔끔한 메시지 도착!
```
